Арифметичні оператори

C++ підтримує стандартні арифметичні оператори для роботи з числовими типами даних.

ОператорОпераціяПрикладРезультат
+Додавання5 + 38
-Віднімання5 - 32
*Множення5 * 315
/Ділення5 / 31 (цілочисельне)
%Остача від ділення5 % 32
⚠️ Важливо: цілочисельне ділення

Якщо обидва операнди цілі, оператор / виконує цілочисельне ділення (дробова частина відкидається). Наприклад, 5 / 2 = 2, а не 2.5. Для отримання дійсного результату, хоча б один операнд має бути дійсного типу: 5.0 / 2 = 2.5.

Скорочені оператори присвоєння:

ОператорЕквівалентПриклад
+=a = a + ba += 5
-=a = a - ba -= 3
*=a = a * ba *= 2
/=a = a / ba /= 4
%=a = a % ba %= 7

Арифметичні вирази

Вираз — це комбінація операндів (змінних, констант) та операторів, яка обчислюється в певне значення.

Приклади виразів
// Прості вирази
int a = 10, b = 3;
int sum = a + b;        // 13
int diff = a - b;       // 7
int prod = a * b;       // 30
int quot = a / b;       // 3 (цілочисельне ділення!)
int rem = a % b;        // 1 (остача від 10/3)

// Складні вирази
int x = 5, y = 2, z = 3;
int result = x + y * z;         // 5 + 6 = 11 (спочатку множення!)
double dresult = (double)x / y; // 2.5 (явне перетворення)

// Вираз у виразі
int area = (a + b) * (a - b);   // 13 * 7 = 91

Пріоритет арифметичних операторів

Оператори в C++ мають визначений пріоритет виконання. Оператори з вищим пріоритетом виконуються раніше.

ПріоритетОператориОписПорядок
1 (найвищий)()Круглі дужкиЛіворуч → праворуч
2++, -- (префіксні)Інкремент, декрементЛіворуч → праворуч
3*, /, %Множення, ділення, остачаЛіворуч → праворуч
4+, -Додавання, відніманняЛіворуч → праворуч
5=, +=, -= тощоПрисвоєнняПраворуч → ліворуч
🔑 Правило

Якщо у вас є сумніви щодо порядку обчислень — завжди використовуйте дужки (). Вони не лише гарантують правильний порядок, але й роблять код зрозумілішим для читачів.

Інкремент та декремент

Інкремент (++) збільшує значення змінної на 1, декремент (--) зменшує на 1.

Префіксна форма: спочатку зміна, потім використання

Префіксна форма
int a = 5;
int b = ++a;   // спочатку a = 6, потім b = 6
// Результат: a = 6, b = 6

Постфіксна форма: спочатку використання, потім зміна

Постфіксна форма
int a = 5;
int b = a++;   // спочатку b = 5, потім a = 6
// Результат: a = 6, b = 5
✅ Рекомендація

У складних виразах використовуйте інкремент/декремент окремо від інших операцій, щоб уникнути плутанини та невизначеної поведінки. Наприклад, замість arr[i++] = arr[++j] краще розбити на кілька інструкцій.

Математичні функції

Для використання математичних функцій необхідно підключити заголовок <cmath>.

ФункціяОписПриклад
pow(x, y)x у степені ypow(2, 3) = 8
sqrt(x)Квадратний коріньsqrt(16) = 4
cbrt(x)Кубічний коріньcbrt(27) = 3
abs(x)Модуль (цілий)abs(-5) = 5
fabs(x)Модуль (дійсний)fabs(-3.14) = 3.14
sin(x)Синус (x у радіанах)sin(0) = 0
cos(x)Косинусcos(0) = 1
tan(x)Тангенсtan(0) = 0
log(x)Натуральний логарифмlog(2.718) ≈ 1
log10(x)Десятковий логарифмlog10(100) = 2

ceil(x)Округлення вгоруceil(3.2) = 4
floor(x)Округлення внизfloor(3.9) = 3
round(x)Округлення до найближчогоround(3.5) = 4
Приклад використання cmath
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    double a = 16.0;
    double b = 3.0;
    
    cout << "sqrt(16) = " << sqrt(a) << endl;
    cout << "pow(2, 3) = " << pow(2.0, 3.0) << endl;
    cout << "fabs(-3.14) = " << fabs(-3.14) << endl;
    cout << "ceil(3.2) = " << ceil(3.2) << endl;
    cout << "floor(3.9) = " << floor(3.9) << endl;
    cout << "round(3.5) = " << round(3.5) << endl;
    
    return 0;
}

Формати виводу даних

Для форматованого виведення даних у C++ використовуються спеціальні функції та маніпулятори з бібліотеки <iomanip>.

Основні функції форматування:

Маніпулятор/функціяОписПриклад
setw(n)Встановлює ширину поля виведення в n символівcout << setw(10) << 42;
setprecision(n)Встановлює кількість значущих цифрcout << setprecision(4) << 3.14159;
fixedФіксований формат дійсних чиселcout << fixed << 3.14159;
scientificНаукова нотаціяcout << scientific << 123.45;
leftВирівнювання по лівому краюcout << left << setw(10) << "text";
rightВирівнювання по правому краюcout << right << setw(10) << 42;
setfill(ch)Символ заповненняcout << setfill('0') << setw(5) << 42;
showpoint / noshowpointПоказувати/приховувати десяткову крапкуcout << showpoint << 5.0;

Маніпулятори виведення: приклади

Форматований вивід
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    double pi = 3.14159265359;
    int number = 42;
    
    // Ширина поля та вирівнювання
    cout << "Без форматування: [" << number << "]" << endl;
    cout << "Ширина 10:        [" << setw(10) << number << "]" << endl;
    cout << "Ліве вирівнювання:[" << left << setw(10) << number << "]" << endl;
    cout << "Заповнення '0':   [" << setfill('0') << setw(5) << number << "]" << endl;
    
    // Форматування дійсних чисел
    cout << "\nТочність 4:       " << setprecision(4) << pi << endl;
    cout << "Фіксований:       " << fixed << setprecision(3) << pi << endl;
    cout << "Науковий:         " << scientific << pi << endl;
    
    // Таблиця
    cout << "\n--- Таблиця товарів ---" << endl;
    cout << setfill(' ') << left;
    cout << setw(15) << "Назва" << setw(10) << "Ціна" << setw(10) << "Кількість" << endl;
    cout << setw(15) << "Яблука" << setw(10) << 25.50 << setw(10) << 10 << endl;
    cout << setw(15) << "Груші" << setw(10) << 35.00 << setw(10) << 5 << endl;
    
    return 0;
}

Результат:

Результат
Без форматування: [42]
Ширина 10:        [        42]
Ліве вирівнювання:[42        ]
Заповнення '0':   [00042]

Точність 4:       3.142
Фіксований:       3.142
Науковий:         3.142e+00

--- Таблиця товарів ---
Назва          Ціна      Кількість
Яблука         25.5      10
Груші          35        5
✅ Порада

Для виведення грошових сум завжди використовуйте fixed разом з setprecision(2), щоб гарантувати два знаки після коми.

💻 Практичні завдання

1Калькулятор з меню

Напишіть програму-калькулятор, яка зчитує два числа та операцію (+, -, *, /, %) і виводить результат з точністю 4 знаки після коми.

2Форматована таблиця чисел

Виведіть таблицю квадратів та кубів чисел від 1 до 20 у форматі: | Номер | Квадрат | Куб | з вирівнюванням та шириною стовпців 10 символів.

3Розв'язання квадратного рівняння

Напишіть програму для розв'язання ax² + bx + c = 0. Обробіть всі випадки (D>0, D=0, D<0). Виведіть результати з точністю 3 знаки.

4Конвертер одиниць

Напишіть програму для конвертації: дюйми→см, фунти→кг, милі→км, градуси→радіани. Використайте форматований вивід у вигляді таблиці.

5Обчислення виразу

Обчисліть значення виразу: (a+b)²/(c-d) + √(a²+b²) - |a-b| для введених a, b, c, d. Обробіть випадок ділення на нуль.

6Відсотки та знижки

Напишіть програму, яка обчислює ціну товару зі знижкою. Введіть початкову ціну та відсоток знижки. Виведіть: економію та кінцеву ціну.

7Геометричні обчислення

Обчисліть площу та периметр прямокутника, площу кола та довжину кола, об'єм кулі. Виведіть результати з точністю 2 знаки після коми.

8Фізичні розрахунки

Напишіть програму для обчислення: шляху при рівноприскореному русі s = v₀t + at²/2, кінетичної енергії E = mv²/2, сили тяжіння F = mg.

9Перевірка на парність

Напишіть програму, яка визначає, чи є число парним або непарним, використовуючи оператор %.

10Інверсія числа

Напишіть програму, яка інвертує (перевертає) тризначне число. Наприклад: 123 → 321. Використайте оператори / та %.
