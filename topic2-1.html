Що таке програма

Програма — це послідовність інструкцій (команд), написаних мовою програмування, які виконуються комп'ютером для розв'язання певної задачі.

Програма на C++ складається з наступних елементів:

  • Директиви препроцесора — команди, що виконуються до компіляції
  • Оголошення глобальних об'єктів — змінні, функції, класи
  • Функція main() — головна функція, з якої починається виконання
  • Інструкції та вирази — команди для виконання
  • Коментарі — пояснення для програміста (ігноруються компілятором)
🔑 Основне правило

Будь-яка програма на C++ повинна містити функцію main(). Виконання програми завжди починається з цієї функції, незалежно від її розташування у файлі.

Структура програми на C++

Загальна структура програми на C++ має такий вигляд:

Структура програми C++
// 1. Директиви препроцесора
#include <iostream>

// 2. Оголошення простору імен
using namespace std;

// 3. Глобальні константи та змінні (необов'язково)
const double PI = 3.14159;

// 4. Прототипи функцій (якщо потрібно)
void myFunction();

// 5. Головна функція
int main() {
    // Тіло функції main — інструкції програми
    
    return 0; // Код завершення програми
}

Кожна інструкція в C++ завершується символом ; (крапка з комою). Цей символ є обов'язковим і позначає кінець інструкції.

Перша програма: "Hello, World!"

Традиційно перша програма, яку пишуть при вивченні нової мови програмування, виводить на екран повідомлення "Hello, World!". Розглянемо її:

hello.cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

Розберемо цю програму по рядках:

  • #include <iostream> — директива препроцесора, що підключає бібліотеку для введення/виведення
  • using namespace std; — використання стандартного простору імен
  • int main() — оголошення головної функції, що повертає ціле число
  • cout << "Hello, World!" << endl; — виведення тексту на екран
  • return 0; — повернення коду завершення 0 (успішне завершення)

Введення та виведення даних

У C++ введення/виведення здійснюється через потоки (stream):

Об'єкти для роботи з потоками:

Об'єктПризначенняПриклад
coutВиведення на екран (console output)cout << "Текст";
cinВведення з клавіатури (console input)cin >> змінна;
cerrВиведення помилокcerr << "Помилка!";
clogВиведення логівclog << "Лог";

Оператори потоків:

  • << — оператор вставки в потік (виведення)
  • >> — оператор вилучення з потоку (введення)

Приклад програми з введенням даних:

Введення та виведення
#include <iostream>
using namespace std;

int main() {
    string name;
    int age;
    
    cout << "Введіть ваше ім'я: ";
    cin >> name;
    
    cout << "Введіть ваш вік: ";
    cin >> age;
    
    cout << "Привіт, " << name << "!" << endl;
    cout << "Вам " << age << " років." << endl;
    
    return 0;
}

Маніпулятори виведення:

МаніпуляторПризначення
endlПерехід на новий рядок та очищення буфера
setw(n)Встановлення ширини поля виведення
setprecision(n)Встановлення кількості знаків після коми
fixedФіксований формат для дійсних чисел
left / rightВирівнювання по лівому/правому краю

Коментарі

Коментарі — це текст у програмі, який ігнорується компілятором. Вони використовуються для пояснення коду.

В C++ є два типи коментарів:

Типи коментарів
// Це однорядковий коментар

/* Це багаторядковий коментар.
   Він може займати кілька рядків.
   Зазвичай використовується для опису функцій */

/*
 * Опис функції: обчислює суму двох чисел
 * Параметри: a, b — доданки
 * Повертає: суму a + b
 */
int sum(int a, int b) {
    return a + b; // повертаємо результат
}
✅ Рекомендації щодо коментарів
  • Коментуйте складні алгоритми та неочевидні рішення
  • Не коментуйте очевидний код — код має бути зрозумілим сам по собі
  • Використовуйте коментарі для документування функцій та класів
  • Підтримуйте коментарі в актуальному стані при зміні коду

Директиви препроцесора

Препроцесор — це програма, що обробляє вихідний код перед компіляцією. Всі директиви препроцесора починаються з символу #.

Основні директиви:

ДирективаПризначенняПриклад
#includeПідключення заголовкових файлів#include <iostream>
#defineВизначення макросу#define PI 3.14
#ifdefУмовна компіляція#ifdef DEBUG
#ifndefУмовна компіляція (якщо не визначено)#ifndef HEADER_H
#endifКінець умовної компіляції#endif
#pragmaСпеціальні команди компілятору#pragma once

Дві форми директиви #include:

  • #include <файл> — для стандартних бібліотечних файлів (шукає в системних каталогах)
  • #include "файл" — для власних заголовкових файлів (шукає спочатку в поточному каталозі)

Простір імен (namespace)

Простір імен — це механізм C++ для організації коду в логічні групи та запобігання конфліктам імен.

Стандартна бібліотека C++ міститься у просторі імен std. Для доступу до його елементів є три способи:

Простір імен
// Спосіб 1: using namespace — використовувати весь простір
using namespace std;
cout << "Hello" << endl;

// Спосіб 2: using — підключити конкретний елемент
using std::cout;
using std::endl;
cout << "Hello" << endl;

// Спосіб 3: Повне кваліфіковане ім'я (найбезпечніше)
std::cout << "Hello" << std::endl;
⚠️ Попередження

Використання using namespace std; у великих проєктах не рекомендується, оскільки може призвести до конфліктів імен. Краще використовувати повне кваліфіковане ім'я std:: або підключати тільки потрібні елементи.

💻 Практичні завдання

1Hello, World! з ім'ям

Напишіть програму, яка запитує ім'я користувача та виводить: "Привіт, [ім'я]! Ласкаво просимо до світу C++!"

2Калькулятор двох чисел

Напишіть програму, яка зчитує два цілі числа з клавіатури та виводить їх суму, різницю, добуток та частку.

3Прямокутник із зірочок

Напишіть програму, яка виводить прямокутник розміром m×n з символу '*', де m та n вводить користувач.

4Таблиця множення

Напишіть програму, яка виводить таблицю множення для числа, введеного користувачем (від 1 до 10).

5Конвертер температури

Напишіть програму для конвертації температури з Цельсія в Фаренгейт та Кельвін. Формули: F = C×9/5+32, K = C+273.15

6Обмін значень

Напишіть програму, яка зчитує два числа, міняє їх місцями та виводить результат. Реалізуйте двома способами: з допоміжною змінною та без неї.

7Відстань між точками

Напишіть програму, яка обчислює відстань між двома точками на площині за координатами (x1,y1) та (x2,y2). Формула: d = sqrt((x2-x1)²+(y2-y1)²)

8Час у секундах

Напишіть програму, яка переводить час у форматі години:хвилини:секунди в загальну кількість секунд.

9Середнє арифметичне

Напишіть програму, яка зчитує три дійсні числа та обчислює їх середнє арифметичне та середнє геометричне.

10Площа та периметр

Напишіть програму, яка обчислює площу та периметр: а) прямокутника, б) трикутника (за трьома сторонами через формулу Герона), в) кола.
