Поняття багатовимірного масиву

Багатовимірний масив — це масив, елементами якого є інші масиви. Найчастіше використовуються двовимірні масиви (матриці), які представляють дані у вигляді таблиць.

Приклади використання:

  • Матриці в лінійній алгебрі
  • Ігрові дошки (шахи, хрестики-нулики)
  • Пікселі зображення
  • Таблиці баз даних

Оголошення та ініціалізація

Двовимірний масив
// Оголошення: тип ім'я[рядки][стовпці];
int matrix[3][4];           // 3 рядки, 4 стовпці
double table[10][10];       // 10x10

// Ініціалізація
int m1[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};

// Часткова ініціалізація
int m2[3][3] = {
    {1, 2},
    {3},
    {4, 5, 6}
};
// Результат:
// 1 2 0
// 3 0 0
// 4 5 6

// Доступ до елементів
int val = m1[0][2];     // 3 (рядок 0, стовпець 2)
m1[1][0] = 100;         // зміна елемента

Двовимірні масиви (матриці)

Введення та виведення матриці
#include <iostream>
using namespace std;

int main() {
    const int ROWS = 3, COLS = 4;
    int matrix[ROWS][COLS];
    
    // Введення матриці
    cout << "Введіть матрицю " << ROWS << "x" << COLS << ":" << endl;
    for (int i = 0; i < ROWS; i++) {
        for (int j = 0; j < COLS; j++) {
            cin >> matrix[i][j];
        }
    }
    
    // Виведення матриці
    cout << "Матриця:" << endl;
    for (int i = 0; i < ROWS; i++) {
        for (int j = 0; j < COLS; j++) {
            cout << matrix[i][j] << "\t";
        }
        cout << endl;
    }
    
    return 0;
}

Операції з матрицями

Операції з матрицями
const int N = 3;

// Сума діагональних елементів
int diagSum = 0;
for (int i = 0; i < N; i++) diagSum += a[i][i];

// Сума елементів кожного рядка
for (int i = 0; i < N; i++) {
    int rowSum = 0;
    for (int j = 0; j < N; j++) rowSum += a[i][j];
    cout << "Sum row " << i << ": " << rowSum << endl;
}

// Транспонування матриці
for (int i = 0; i < N; i++) {
    for (int j = i + 1; j < N; j++) {
        int temp = a[i][j];
        a[i][j] = a[j][i];
        a[j][i] = temp;
    }
}

// Додавання двох матриць
for (int i = 0; i < N; i++)

        c[i][j] = a[i][j] + b[i][j];

// Множення матриць

    for (int j = 0; j < N; j++) {
        c[i][j] = 0;

Багатовимірні масиви у функціях

Масиви у функціях
// Функція для друку матриці (треба вказати розмір стовпця!)
void printMatrix(int mat[][100], int rows, int cols) {
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < cols; j++)
            cout << mat[i][j] << "\t";
        cout << endl;
    }
}

// Сума головної діагоналі
int trace(int mat[][100], int n) {
    int sum = 0;
    for (int i = 0; i < n; i++)
        sum += mat[i][i];
    return sum;
}

// Перевірка чи матриця симетрична
bool isSymmetric(int mat[][100], int n) {
    for (int i = 0; i < n; i++)
        for (int j = i + 1; j < n; j++)
            if (mat[i][j] != mat[j][i])
                return false;
    return true;
}

💻 Практичні завдання

1Сума рядків та стовпців

Напишіть програму, яка обчислює суму елементів кожного рядка та кожного стовпця матриці MxN.

2Пошук сідлової точки

Знайдіть сідлову точку матриці — елемент, який є мінімумом у своєму рядку та максимумом у своєму стовпці.

3Матриця в оберненому порядку

Виведіть елементи матриці NxN по спіралі: з початку за годинниковою стрілкою.

4Змійка

Заповніть матрицю NxN числами від 1 до N² у вигляді змійки: перший рядок зліва направо, другий справа наліво тощо.

5Визначник матриці 3x3

Напишіть функцію для обчислення визначника матриці 3x3 за правилом Саррюса.

6Матриця суміжності

Створіть матрицю суміжності для графа з N вершин та M ребер. Реалізуйте виведення ступенів вершин.

7Обнулення нижнього трикутника

Напишіть функцію, яка обнує всі елементи нижче головної діагоналі (перетворення у верхню трикутну матрицю).

8Гра "Хрестики-нулики"

Реалізуйте гру хрестики-нулики на полі 3x3 з виведенням дошки та перевіркою переможця.

9Звичайний квадрат

Перевірте, чи є матриця NxN магічним квадратом (суми по рядках, стовпцях та діагоналях рівні).

10Обертання матриці на 90°

Напишіть функцію, яка обертає квадратну матрицю на 90 градусів за годинниковою стрілкою на місці (без додаткової матриці).
