C-style строки (символьні масиви)

C-style строка — це масив символів, що завершується нульовим символом '\0'.

C-style строки
// Оголошення C-style строки
char str1[20] = "Hello";      // 5 символів + '\0'
char str2[] = "World";        // розмір визначається автоматично (6)
char str3[20];

// Введення строки
 cin >> str3;        // зчитує до пробілу
 cin.getline(str3, 20);  // зчитує рядок включно з пробілами

// Довжина строки
int len = strlen(str1);   // 5

C++ string клас (заголовок <string>)

Клас string з стандартної бібліотеки C++ є набагато зручнішим та безпечнішим за C-style строки.

C++ string
#include <string>
using namespace std;

string s1 = "Hello";           // ініціалізація
string s2("World");            // конструктор
string s3 = s1 + " " + s2;   // конкатенація: "Hello World"
string s4;                     // пуста строка

// Введення/виведення
getline(cin, s4);              // зчитування рядка з пробілами
cout << s1 << endl;

// Довжина
int len = s1.length();         // або s1.size()

Строки у функціях

Строки у функціях
// Передача string за значенням (копія)
void printString(string s) {
    cout << s << endl;
}

// Передача за константним посиланням (рекомендовано)
void printString(const string& s) {
    cout << s << endl;
}

// Повернення string
string toUpperCase(const string& s) {
    string result = s;
    for (int i = 0; i < result.length(); i++) {
        result[i] = toupper(result[i]);
    }
    return result;
}

// Функція перевірки паліндрому
bool isPalindrome(const string& s) {
    int left = 0, right = s.length() - 1;
    while (left < right) {
        if (s[left] != s[right]) return false;
        left++;
        right--;
    }
    return true;
}

💻 Практичні завдання

1Підрахунок слів

Напишіть функцію, яка підраховує кількість слів у рядку. Словом вважається послідовність символів, відмінних від пробілу.

2Перевертання рядка

Напишіть функцію, яка перевертає рядок (реверс). Реалізуйте для C-string та C++ string.

3Підрахунок голосних

Напишіть функцію, яка підраховує кількість голосних літер у рядку (українські та англійські).

4Заміна символів

Напишіть функцію replaceAll(s, from, to), яка замінює всі входження підстроки from на to у рядку s.

5Валідація email

Напишіть функцію, яка перевіряє, чи є рядок коректною email-адресою (наявність @ та домену).

6Caesar cipher

Реалізуйте шифр Цезаря для рядків (зсув на k позицій). Зробіть функції шифрування та дешифрування.

7Анаграма

Напишіть функцію, яка перевіряє, чи є два рядки анаграмами (містять однакові літери в різному порядку).

8Перетворення регістру

Реалізуйте функції toUpper та toLower для рядків. Обробіть як латиницю, так і кирилицю.

9Парсер чисел з рядка

Напишіть функцію, яка зчитує всі числа з рядка та зберігає їх у масив (наприклад: "12, 45, 78" → [12, 45, 78]).

10Бібліотека StringUtils

Створіть модуль stringutils.h/cpp з функціями: trim, split, join, startsWith, endsWith, contains, reverse. Протестуйте кожну функцію.
