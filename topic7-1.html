Поняття структури

Структура (struct) — це користувацький тип даних, що об'єднує змінні різних типів під одним іменем. Структури використовуються для представлення складних об'єктів реального світу.

🔑 Ключова ідея

Структура дозволяє групувати логічно пов'язані дані. Наприклад, студент має ім'я, вік, номер групи та середній бал — все це можна об'єднати в одну структуру.

Оголошення структури

Структура в C++
struct Student {
    string name;       // ім'я
    int age;           // вік
    string group;      // група
    double avgGrade;   // середній бал
};

// Створення змінної-структури
Student st1;
st1.name = "Іван Петренко";
st1.age = 20;
st1.group = "КН-21";
st1.avgGrade = 4.5;

// Ініціалізація при оголошенні
Student st2 = {"Марія Коваленко", 19, "КН-22", 4.8};

// Список ініціалізації (C++11)
Student st3{"Олег Сидоров", 21, "КН-21", 3.9};

Доступ до членів структури

Доступ до членів
// Оператор . (крапка) — доступ через об'єкт
Student s;
s.name = "Anna";
cout << s.age;

// Оператор -> (стрілка) — доступ через вказівник
Student* ptr = &s;
ptr->name = "Anna";     // еквівалентно (*ptr).name
cout << ptr->age;

Вкладені структури

Вкладені структури
struct Date {
    int day;
    int month;
    int year;
};

struct Person {
    string name;
    Date birthDate;  // вкладена структура
    string address;
};

Person p;
p.name = "Іван";
p.birthDate.day = 15;
p.birthDate.month = 3;
p.birthDate.year = 2000;

Структури у функціях

Структури у функціях
// Передача структури в функцію
void printStudent(const Student& s) {
    cout << "Ім'я: " << s.name << endl;
    cout << "Вік: " << s.age << endl;
    cout << "Група: " << s.group << endl;
    cout << "Середній бал: " << s.avgGrade << endl;
}

// Повернення структури
Student createStudent(string n, int a, string g, double grade) {
    Student s{n, a, g, grade};
    return s;
}

// Функція порівняння
bool compareByGrade(const Student& a, const Student& b) {
    return a.avgGrade > b.avgGrade;
}

typedef та using

Псевдоніми типів
// typedef — створення псевдоніму типу
typedef struct {
    double x;
    double y;
} Point;

// using — сучасний спосіб (C++11)
using Point2D = struct {
    double x;
    double y;
};

// Використання
Point p1 = {3.0, 4.0};
Point2D p2 = {1.0, 2.0};

💻 Практичні завдання

1Книжкова бібліотека

Створіть структуру Book (назва, автор, рік, кількість сторінок). Створіть функції для введення, виведення та пошуку книги за автором.

2Геометричні фігури

Створіть структури для точки, кола та прямокутника. Реалізуйте функції обчислення площі та перевірки перетину.

3База даних студентів

Створіть структуру Student та меню: додати, видалити, знайти, сортувати за балом, вивести всіх.

4Час та дата

Створіть структуру Time (год, хв, сек) та Date. Реалізуйте функцію додавання часу та різниці між датами.

5Комплексні числа

Створіть структуру Complex. Реалізуйте додавання, віднімання, множення та ділення комплексних чисел.

6Точка на площині

Створіть структуру Point з координатами x, y. Реалізуйте функції обчислення відстані до центру, відстані між точками, перевірки чи лежить на прямій.

7Співробітник компанії

Створіть структуру Employee з вкладеною Address. Реалізуйте пошук за прізвищем та сортування за зарплатою.

8RGB колір

Створіть структуру Color (r, g, b). Реалізуйте функції змішування кольорів та перетворення у grayscale.

9Матриця через структуру

Створіть структуру Matrix з двовимірним масивом та розмірами. Реалізуйте додавання, множення та транспонування.

10Адресна книга

Створіть програму адресної книги з використанням структури Contact. Збереження/завантаження з файлу, пошук, сортування.
